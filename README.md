# Fertility-Insights-IVF-Operations
## project overview
This project analyzes IVF clinic operational datasets to identify inefficiencies in patient wait times, staffing, room utilization, equipment usage, and operational costs
## Objectives
- Clean and preprocess IVF operational data
- Calculate descriptive statistics (mean, median, mode)
- Perform univariate and bivariate analysis
- Visualize patterns and correlations
- Support data-driven operational decisions
## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
-  mysql,powerbi
  ## Key Insights
- High patient wait times correlate with staff and room utilization
- Uneven scheduling increases operational costs
- Efficient resource allocation improves performance KPIs
